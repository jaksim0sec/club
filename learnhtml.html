html 이란?

HTML 은 웹사이트의 기본 뼈대가 되는 언어입니다. HTML 을 이용하여 텍스트, 버튼, 이미지 등등 여러가지 "요소" 를 생성할 수 있습니다.

html 기본 베이스

HTML 의 요소는 태그를 이용하여 생성합니다.
HTML 의 태그는 < 태그종류(예: 이미지태그) > < /태그이름> 과 같은 식으로 써야합니다.
태그는 프로그래머가 기입한 태그 순서대로 위에서 아래, 왼쪽에서 오른쪽 방향으로 생성됩니다.
이들은 실행을 위해 일정한 규칙대로 작성되어야합니다. 양식은 아래와 같습니다.
<!DOCTYPE html>
          <html lang="en">
          <head>
          <meta charset="UTF-8">
          <meta name="viewport"
          content="width=device-width, initial-scale=1.0">
          <title>문서 제목</title>
          </head>
          <body>
          <!-- 내용을 이곳에 추가합니다 -->
          </body>
          </html>
매우 복잡해보일 수 있지만 자세히 보면 간단합니다. 이제 자세히 알아보겠습니다.

<!DOCTYPE html>

컴퓨터에게 아래부터는 html 언어를 사용하겠다고 알려주는 태그 입니다!

<html> </html>

<html> 와 </html> 사이에 html 코드를 작성해야합니다.

<head> </head>

<head> 와 </head> 사이에는 사이트의 파비콘과 제목이 표시되는 부분을 꾸미는 태그를 넣습니다.
▲ 이 부분을 꾸민다는 뜻입니다.
꾸미는 코드의 예시 태그로는 <title> </title> 이 있습니다. 두 요소 사이에 들어가는 글자를 사이트 제목으로 삼습니다.
예를 들어 이 사이트는 <title>웹개발부 개념설명과 해보기 </title> 라고 작성되어 있습니다.

<meta charset="UTF-8">
<meta name="viewport"

웹사이트가 잘 돌아가도록 해주는 기본 설정 태그인데, 이건 회장도 안외웁니다. 이딴거 외우지 마세요. head 사이에 들어가 있는 구조입니다.
▲ (이해를 돕기 위해) 이 사이트의 head 는 이렇게 구성되어있습니다.

<body> </body>

위 이미지처럼 실제 사용자가 사용하는, 그러니까 개발자가 사용자에게 기능을 제공하는 부분을 꾸미는 태그가 들어가는 태그입니다.
이제 body 에서 유용하게 사용할 수 있는 태그를 알아보겠습니다!

<h> </h> 태그

지금 이 글을 쓸때도 h 태그를 사용했어요.
태그는 <h> 글자 내용 </h> 의 모양꼴로 사용합니다. h뒤에는 숫자를 붙여야 하는데, 숫자의 크기가 커질수록 글자의 크기가 작아저요! 최대 h6 까지 가능해요.

<a> </a> 태그

a태그는 클릭하면 대상 링크로 접속시켜주는 버튼을 만드는 태그에요!
