우리 동아리의 목적

우리 동아리에서 웹개발부는 기본적으로 웹개발사이트를 개발할 수 있도록 학생들을 교육한 이후에, 자신의 개성을 표현한 사이트를 구현하여 샛별제 때 정한 주제의 사이트를 공개할 예정입니다.

웹사이트란?

웹사이트가 뭔지는 보통 아실겁니다. 네이버나 구글, 심지어 지금 접속하시고 계신 회장이 만든 웹페이지 까지, 모두 웹사이트 입니다.

웹사이트는 html, js, css 로 이루어졌는데, 각각 사이트의 뼈, 장기와 뇌, 피부 에 해당합니다.

html 이란?

모든 웹사이트는 html 라는 프로그래밍 언어를 베이스로 제작됩니다. 웹사이트를 인간으로 비유하면 html 는 뼈에 해당하죠. 물론 이 웹사이트도 html 를 사용했어요

html 기본 베이스

html 은 "요소" 라는 것을 위에서 아래 방향으로 순서대로 끼우듯 작성하는 언어입니다. 요소는 예를 들면 "이미지파일","글자(지금 이 글도 포함)","버튼" 과 같은 것들 입니다.
html 의 요소를 선언하기 위해서는 < 태그이름 > < /태그이름> 과 같은 식으로 써야합니다.
이들은 실행을 위해 일정한 규칙대로 작성되어야합니다. 양식은 아래와 같습니다.
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport"
content="width=device-width, initial-scale=1.0">
<title>문서 제목</title>
</head>
<body>
<!-- 내용을 이곳에 추가합니다 -->
</body>
</html>
매우 복잡해보일 수 있지만 자세히 보면 간단합니다. 이제 자세히 알아보겠습니다.

<!DOCTYPE html>

컴퓨터에게 아래부터는 html 언어를 사용하겠다고 알려주는 태그 입니다!

<html> </html>

<html> 와 </html> 사이에 html 코드를 작성해야합니다.

<head> </head>

<head> 와 </head> 사이에는 사이트의 파비콘과 제목이 표시되는 부분을 꾸미는 태그를 넣습니다.
▲ 이 부분을 꾸민다는 뜻입니다.
꾸미는 코드의 예시 태그로는 <title> </title> 이 있습니다. 두 요소 사이에 들어가는 글자를 사이트 제목으로 삼습니다.
예를 들어 이 사이트는 <title>웹개발부 개념설명과 해보기 </title> 라고 작성되어 있습니다.

<meta charset="UTF-8">
<meta name="viewport"

웹사이트가 잘 돌아가도록 해주는 기본 설정 태그인데, 이건 회장도 안외웁니다. 이딴거 외우지 마세요. head 사이에 들어가 있는 구조입니다.
▲ (이해를 돕기 위해) 이 사이트의 head 는 이렇게 구성되어있습니다.

<body> </body>

위 이미지처럼 실제 사용자가 사용하는, 그러니까 개발자가 사용자에게 기능을 제공하는 부분을 꾸미는 태그가 들어가는 태그입니다.
이제 body 에서 유용하게 사용할 수 있는 태그를 알아보겠습니다!

<h> </h> 태그

지금 이 글을 쓸때도 h 태그를 사용했어요.
