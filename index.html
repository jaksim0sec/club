html 이란?

HTML 은 웹사이트의 기본 뼈대가 되는 언어입니다. HTML 을 이용하여 텍스트, 버튼, 이미지 등등 여러가지 "요소" 를 생성할 수 있습니다.

html 기본 베이스

HTML 의 요소는 태그를 이용하여 생성합니다.
HTML 의 태그는 < 태그종류(예: 이미지태그) > < /태그이름> 과 같은 식으로 써야합니다.
태그는 프로그래머가 기입한 태그 순서대로 위에서 아래, 왼쪽에서 오른쪽 방향으로 생성됩니다.
이들은 실행을 위해 일정한 규칙대로 작성되어야합니다. 양식은 아래와 같습니다.
<!DOCTYPE html>
            <html lang="en">
            <head>
            <meta charset="UTF-8">
            <meta name="viewport"
            content="width=device-width, initial-scale=1.0">
            <title>문서 제목</title>
            </head>
            <body>
            <!-- 내용을 이곳에 추가합니다 -->
            </body>
            </html>
매우 복잡해보일 수 있지만 자세히 보면 간단합니다. 이제 자세히 알아보겠습니다.

<!DOCTYPE html>

<html> </html>

<html> 와 </html> 사이에 html 코드를 작성해야합니다.

<head> </head>

<head> 와 </head> 사이에는 사이트의 파비콘과 제목이 표시되는 부분을 꾸미는 태그를 넣습니다.
▲ 이 부분을 꾸민다는 뜻입니다.
꾸미는 코드의 예시 태그로는 <title> </title> 이 있습니다. 두 요소 사이에 들어가는 글자를 사이트 제목으로 삼습니다.
예를 들어 이 사이트는 <title>웹개발부 개념설명과 해보기 </title> 라고 작성되어 있습니다.

<meta charset="UTF-8">
<meta name="viewport"

JavaScript 란?

JavaScript (js) 는 웹페이지를 동적으로 만들거나 요소를 생성, 관리, 삭제, 수정 할 수 있는 프로그래밍 언어입니다.

JS 는 html 에서 <script&gpt; 와 </script> 사이에 작성됩니다.

출력

브라우저의 개발자 도구에는 주로 콘솔창이라고 불리는 도구가 있습니다. 여기에는 JavaScript 코드의 결과나 console.log()에서 출력한 메시지 등이 표시됩니다.

console.log() 는 () 속의 값을 출력합니다.

변수

일반적으로 변수는 숫자나 글자, 또는 정보를 저장하고 참조할 수 있는 공간입니다.

변수는 각각의 이름을 가지고 있으며, 이후 이름을 통해 다시 참조할 수 있습니다.

( "//내용" 을 통하여 주석(메모) 를 작성 가능


              // 변수 선언
              let 변수1;
              // 변수에 값 할당
              변수1 = 5;
              // 변수 초기화 (변수선언을 할때 한방에 이렇게 해도 됨)
              let 변수2 = 10;
              // 변수 참조
              console.log(변수1 + 변수2); // 15 출력
              //일반적으로 변수 이름은 영어와 언더바를 혼합해서 사용.
              //텍스트 코딩을 할때에는 숫자값과 글자값을 정확히 구분해야합니다.
              //코드속에서 연산이 되는 숫자 / 숫자값을 가지고 있는 변수 는 "" 를 사용하면 안됩니다. 반대로 연산이 불가능한 글자값 / 글자값을 가지고 있는 변수는 "" 를

조건문

조건문은 주어진 조건에 따라 코드의 실행을 제어합니다.


              let age = 20;
              if (age >= 18) {
                  console.log("성인입니다.");
              } else {
                  console.log("미성년자입니다.");
              }

반복문

반복문은 주어진 조건에 따라 코드 블록을 반복해서 실행합니다.


              for (let i = 0; i < 5; i++) {
                  console.log("반복문 실행 중:", i);
              }

함수

함수는 코드를 그룹화하고 재사용 가능한 코드 블록을 생성합니다.


              function greet(name) {
                  console.log("안녕하세요, " + name + "님!");
              }
              greet("철수");
              greet("영희");

객체

객체는 관련된 변수와 함수를 그룹화하여 사용합니다.


              let person = {
                  name: "철수",
                  age: 25,
                  greet: function() {
                      console.log("안녕하세요, 저는 " + this.name + "입니다.");
                  }
              };
              console.log(person.name); // "철수"
              person.greet(); // "안녕하세요, 저는 철수입니다."

배열(리스트)

배열은 여러 값을 하나의 변수에 저장하는 데 사용됩니다.


              //배열 선언 또는 수정
              let fruits = ["사과", "바나나", "딸기"];
              //배열 접근
              console.log(fruits[0]); // "사과" - fruit[0] 는 fruits의 첫번째 값을 의미합니다. 배열에서 [] 속에는 "원하는 값의 인덱스 - 1" 의 값를 넣어야 합니다.
              console.log(fruits.length); // 3

여기까지 똑바로 읽으셨다면 이러한 의문이 생기실 수 있습니다.

            document.querySelector("body").style.backgroundColor = "red";

위의 코드를 안다면 html 의 왠만한 접근(요소 수정)은 가능합니다.

위 코드를 한국어 문장으로 억지로 해석하자면,

html 페이지에서 body 라는 이름의 태그의 style 속성중 배경색 (background-color) 을 빨강으로 바꾸겠습니다.

또는 쉽게 바꾸자면

              html에서.이름이(ID 또는 CLSS)인 요소의.스타일에서.배경색 = 빨강;

로 볼 수 있습니다.

document

querySelector(대상)

코드 작성 방식

css 는 실제 프로그래밍 언어라고 하기보다는 html 라는 마크업언어에 선택적으로 디자인적 부분을 수정하는 코드 입니다.
html 요소(태그를 사용해서 만든 오브젝트, 예시로 img 요소나 h 요소)를 선택하고, 디자인을 수정할 수 있습니다.
CSS 를 html 도중에 사용하는 방식 첫번째를 알아보겠습니다. 아래는 html 에서 사용되는 h 태그의 글자색을 빨강색으로 바꿔주는 코드입니다.
<style>
h
{
color : red;
}
</style>
한번 자세히 살펴보겠습니다.

<style> </style>

html 문서 도중에 CSS 를 사용하기 위해 사용하는 태그 입니다. 위에 제시된 코드와 같이 두 태그 사이에 css 코드를 작성합니다.
이러한 방식으로 css 를 작성하면, 따로 css 적용 대상을 지정해줘야 합니다.
지정 대상은 간단합니다.
"대상 태그 이름 {변경할 사항} " 처럼 작성합니다.
위 코드에서도 h(대상 태그 이름) {color:red; (변경할 사항)} 처럼 적용된 것 입니다.
"대상 태그 이름" (정식 용어로는 Selector) 에 태그 이름(예시 : h, a, img, div, hr) 를 입력하면 html 에 있는 모든 대상 태그에 {} 속 css 가 적용됩니다. 예를 들어 위의 코드는 문서의 모든 h 태그들의 글씨색을 붉게 만듭니다.
하지만 적용을 원하는 요소만 적용시키고 싶을 경우 아래와 같이 할 수 있습니다.
css 를 html 에서 사용하는 두번쨰 방법을 알아보겠습니다. 아래 코드는 "div" 라는 html 의 태그 (다목적 상자(직사각형). div 와 /div 사이에 여러 태그를 끼워 넣을 수 있음. 그렇게 하면 파일처럼 묶어 사용할 수 있음) 의 크기를 아랫변길이 100픽셀, 높이 10픽셀 로 만든 후에 배경을 빨간색으로 채워 넣은 예시입니다.
<div style = "background-color :red; width : 100px; height : 10px;">
</div>

css 를 html 에 사용하는 방식을 알아봤다면 이제는 css 자체의 문법을 알아보겠습니다.
